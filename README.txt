App landing pages for QR codes

These pages are ready to host as static files.

How to use
1. Upload this folder to any static web host:
   - Netlify
   - Vercel
   - GitHub Pages
   - Cloudflare Pages
   - your own website

2. After upload, each page will have a public URL such as:
   - https://yourdomain.com/whatsapp.html
   - https://yourdomain.com/grab.html
   - https://yourdomain.com/bolt.html

3. Turn each public page URL into a QR code.

Included app pages
- WhatsApp
- Grab
- Bolt
- 12Go Asia
- Google Maps
- Google Translate
- Wise

Notes
- These are option 2 landing pages: one extra tap after scanning.
- Buttons use official App Store and Google Play listings.
- styles.css keeps the pages visually consistent.
